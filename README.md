# SUBLIME-Reproduction

Reproduction of **"Towards Unsupervised Deep Graph Structure Learning"** (SUBLIME, WWW 2022) by Liu et al.

Paper: [arXiv:2201.06367](https://arxiv.org/abs/2201.06367)

This work was carried out for the MSc course **DSAIT4205 — Fundamental Research in Machine and Deep Learning** at **TU Delft**.

## Requirements

The versions below come from the original paper:

- Python 3.7
- PyTorch 1.7.1
- DGL 0.7.1
- NumPy 1.20.2
- SciPy 1.6.3
- Scikit-learn 0.24.2
- Munkres 1.1.4
- ogb 1.3.1
